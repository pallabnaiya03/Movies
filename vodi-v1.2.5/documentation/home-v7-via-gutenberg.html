TV Show Jumbotron is a static block used as a jumbotron. A TV Show jumbotron is assigned via Vodi > TV Shows > TV Show jumbotron. You can simple de-assign it to remove the block.

Settings

To configure sidebar for Home v7, navigate to Appearance > Widgets > TV Show Sidebar here drag and drop the following widgets

  1. Navigation Menu : Menu Widget for showing the menu.
  2. MasVideos Search Widget : Widget for search the TV Show.
  3. MasVideos Filter Videos by Genre: Display a list of tv show genre to filter tv shows in your site.
